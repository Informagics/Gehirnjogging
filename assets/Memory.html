Memory

  • Beim Memory ist ein 4x4 Spielfeld vorgegeben
  • Für jedes richtige Paar bekommt der Spieler 5 Punkte gutgeschrieben
  • für jedes Falsche Spiel bekommt der Spieler 1 Punkt abgezogen
  • Ziel des Spiels ist es alle Paare aufzudecken
